# مدارس الهمة العلمية الأهلية — alhimma-schools

مستودع بداية المشروع لنظام إدارة مدارس الهمة العلمية الأهلية.

الملف يحتوي على السكافولد المبدئي: صفحة Admin Dashboard مبدئية، نظام تصميم الألوان (gradient أزرق→برتقالي + glow)، وملف مثال لإعداد Firebase.

المالك: ghada20036
التقنيات الافتراضية: Next.js, TypeScript, Firebase (Firestore, Auth, Storage)

تعليمات تشغيل سريعة (محلي):
1. نسخ المشروع
2. تشغيل `npm install`
3. إنشاء ملف `.env.local` بناءً على `firebase.sample.env`
4. تشغيل `npm run dev`

ملاحظة: النص المركزي في الصفحة الرئيسية يجب أن يبقى حرفياً: "إعداد وبرمجة: أ. غادة الحيدري"
